### 4. ✅ Запрет галлюцинаций

**Было:**
```
User: что произошло сегодня в мире, ИИ
Bot: [выдумывает новости про Google, MIT, DeepMind]  ❌
```

**Стало:**
```
User: что произошло сегодня в мире, ИИ
Bot: 🔍 [реальные результаты поиска из интернета]  ✅
```

**Что исправлено:**
```python
async def _handle_llm_generation(self, query: str):
    query_lower = query.lower()
    
    # ЗАПРЕТ ГАЛЛЮЦИНАЦИЙ
    forbidden_topics = [
        'новости', 'что произошло', 'события', 
        'сегодня в', 'актуальн'
    ]
    
    if any(topic in query_lower for topic in forbidden_topics):
        return "🔍 Для актуальной информации используй веб-поиск"
```

**Теперь для новостей ВСЕГДА используется веб-поиск!**

## 🎯 Улучшения стратегии выбора

**Приоритеты стратегий (в порядке проверки):**

1. **file_operation** - если упоминается файл
2. **direct_answer** - если дата/время/вычисления
3. **web_search** - если новости/курсы/актуальное
4. **memory_search** - если вопрос о памяти
5. **imagination** - если просьба придумать
6. **llm_generation** - остальное (БЕЗ галлюцинаций)

```python
async def _determine_strategy(self, query: str):
    query_lower = query.lower()
    
    # 1. ФАЙЛЫ - высший приоритет
    if 'запиши' in query_lower or 'сохрани' in query_lower:
        if '.txt' in query_lower or query_lower == 'сохрани файл':
            return 'file_operation'
    
    # 2. ДАТА/ВРЕМЯ - всегда direct
    if 'какое сегодня' in query_lower or 'время' in query_lower:
        return 'direct_answer'
```

## ⚠️ Важные заметки

### 1. Файлы пользователя
- Создаются в `user_files/user_ID/`
- Только `.txt` файлы (можно расширить)
- Кодировка UTF-8
- Сохраняются навсегда (пока не удалишь вручную)

### 2. Контекст файлов
- Сохраняется в `working_memory`
- Живёт до перезапуска бота
- Команда "сохрани файл" работает только в той же сессии

### 3. Извлечение курсов
- Работает для USD, EUR, BTC
- Ищет паттерны в результатах поиска
- Если не найдено - показывает обычные результаты

### 4. Запрет галлюцинаций
- Действует для: новости, события, "что произошло"
- Бот ПРИНУДИТЕЛЬНО использует веб-поиск
- Нельзя обойти - это защита
