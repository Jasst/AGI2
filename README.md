# 🧠 COGNITIVE AGENT v13 ENHANCED - Документация

## 🎯 Основные улучшения

### 1. ИНТЕЛЛЕКТУАЛЬНЫЙ ВЕБ-ПОИСК

#### Ключевые особенности:

**Автоматическая детекция необходимости поиска:**
- Анализ типа запроса (текущая информация, верификация фактов)
- Проверка уровня уверенности модели
- Оценка актуальности имеющихся знаний
- Детекция маркеров неопределенности в ответах

**Интеллектуальная генерация запросов:**
- Множественные поисковые запросы с приоритетами
- Контекстная адаптация (инструкция, причина, дата)
- Уточняющие запросы на основе пробелов знаний
- Очистка от вопросительных слов

**Анализ пробелов в знаниях:**
```python
class KnowledgeGap:
    topic: str              # О чем не хватает знаний
    reason: str             # Почему обнаружен пробел
    severity: str           # critical/high/medium/low
    search_queries: List[str]  # Что искать
```

#### Когда агент использует поиск:

1. **Критические пробелы** - обнаружены маркеры "не знаю", "не уверен"
2. **Граница знаний** - NEED_SEARCH, OUTDATED, UNKNOWN
3. **Низкая уверенность** - confidence < 0.85
4. **Тип задачи** - CURRENT_INFO, VERIFICATION
5. **Временные маркеры** - "сегодня", "сейчас", "актуально"

### 2. РАСШИРЕННЫЙ КОГНИТИВНЫЙ ЦИКЛ (13 этапов)

```
1. Восприятие запроса
   → Очистка, нормализация

2. Классификация задачи
   → CURRENT_INFO, VERIFICATION + 8 базовых типов

3. Активация памяти
   → 5 типов памяти (включая историю поиска)

4. Оценка достаточности знаний ⭐ НОВОЕ
   → Быстрая проверка: можем ли ответить без поиска?

5. Анализ пробелов знаний ⭐ НОВОЕ
   → Что именно неизвестно? Насколько критично?

6. Интеллектуальный веб-поиск ⭐ НОВОЕ
   → Генерация запросов, выполнение, кэширование

7. Выбор стратегии мышления
   → WEB_RESEARCH + 7 базовых стратегий

8. Планирование рассуждения
   → План шагов для выбранной стратегии

9. Обогащение контекста ⭐ НОВОЕ
   → Интеграция результатов поиска + память

10. Выполнение рассуждения
    → С учетом всей собранной информации

11. Самооценка результата
    → Анализ качества, уверенности, источников

12. Извлечение уроков
    → Что сработало? Что не сработало?

13. Мета-коррекция
    → Адаптация стратегий на будущее
```

## 🔬 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Алгоритм принятия решения о поиске:

```python
def _should_perform_search():
    if any(gap.severity == "critical"):
        return True  # Критические пробелы
    
    if knowledge_boundary in [NEED_SEARCH, OUTDATED, UNKNOWN]:
        return True  # Граница знаний
    
    if initial_confidence < 0.85:
        return True  # Низкая уверенность
    
    if task_type in [CURRENT_INFO, VERIFICATION]:
        return True  # Тип задачи
    
    return False
```

### Приоритизация запросов:

```python
priority = 3  # critical severity
priority = 2  # high severity  
priority = 1  # medium/low severity
```

# При успешном поиске
confidence *= 1.15  # +15%
